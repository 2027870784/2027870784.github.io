*计算机图形学个人项目*

- 在不依赖 OpenGL / DirectX 的前提下，使用 C++ 实现最小可用软件渲染管线。
- 手写向量、矩阵、齐次坐标和基础变换逻辑，完成 Model、View、Projection 变换与屏幕空间映射。
- 基于重心坐标实现三角形光栅化，使用 Z-Buffer 处理像素遮挡关系。
- 实现 Lambert 漫反射光照，并排查模型旋转后法线未同步变换导致的光照异常。
- 使用 CMake 拆分渲染核心与入口程序，提升项目结构清晰度。

# 项目经历

## [从零构建 C++ 3D 软光栅渲染器](./soft-render3d/)

**项目类型：** 计算机图形学个人项目  
**技术栈：** C++、CMake、MVP 变换、重心坐标插值、Z-Buffer、Lambert 漫反射  
**项目角色：** 独立开发

### 项目简介

本项目是在不依赖 OpenGL、DirectX 等高级图形 API 的前提下，使用 C++ 从零实现的软件 3D 渲染器，完成从模型数据到像素图像输出的基础渲染管线。

### 核心功能

- 手写向量、矩阵与齐次坐标相关数学运算
- 实现 MVP 变换、透视除法和屏幕空间映射
- 基于重心坐标完成三角形光栅化
- 使用 Z-Buffer 处理深度测试和遮挡关系
- 实现基础 Lambert 漫反射光照
- 输出 `.ppm` 图像并使用 CMake 组织工程

### 项目亮点

- 从零实践模型到像素的完整软件渲染流程
- 排查黑屏、深度写入、法线变换和输出性能等典型图形学问题
- 通过模块拆分沉淀可继续扩展的渲染核心

## [个人技术门户与自驱动简历系统](./tech-portal-resume/)

**项目类型：** 个人工程实践项目  
**技术栈：** Hexo、Butterfly、Markdown、GitHub Pages、GitHub Actions、Git、YAML  
**项目角色：** 独立开发

### 项目简介

本项目基于 Hexo 静态网站生成器构建个人技术门户，使用 Markdown 管理技术文章、项目经历和简历内容，并通过 GitHub Actions 自动部署到 GitHub Pages。

### 核心功能

- Markdown 技术文章管理
- 项目经历展示
- 在线简历页面
- 响应式页面适配
- GitHub Pages 自动部署
- AI 简历自动化方案探索

### 项目亮点

- 使用静态网站生成器完成从内容到网页的自动转换
- 将个人简历和项目经历沉淀为可持续维护的在线技术资产
- 完成从本地开发、内容编写、静态构建到线上部署的完整工程流程
